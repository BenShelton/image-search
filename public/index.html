Image Search Abstraction Layer Microservice

Usage

To search for an image, append the search terms to the URL:

https://hidden-tor-57160.herokuapp.com/funny

You can also optionally add a page number to get different results:

https://hidden-tor-57160.herokuapp.com/funny?page=2

Results

Searching for an image will give you a list of results.
These will contain information on the images and clickable links.

Extra

To see recently searched for images you can go to Latest Searches
